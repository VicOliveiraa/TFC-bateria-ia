# TFC-bateria-ia

Repositório com os códigos e arquivos de apoio utilizados no Trabalho de Final de Curso sobre monitoramento e estimação de variáveis em baterias de íon-lítio, incluindo o firmware embarcado desenvolvido no PlatformIO e os scripts de análise dos dados experimentais.

## Objetivo

Este repositório foi disponibilizado como material complementar ao TFC, com o objetivo de permitir a consulta dos códigos desenvolvidos ao longo do projeto e apoiar a compreensão da metodologia implementada.

## Conteúdo

Este repositório incluir:

- firmware do sistema embarcado desenvolvido para ESP32 no PlatformIO;
- scripts de aquisição, processamento e organização dos dados;
- rotinas de estimação de variáveis como estado de carga;
- códigos de geração de gráficos e análise de resultados;
- arquivos auxiliares utilizados no desenvolvimento do trabalho.

## Organização

A estrutura do repositório pode incluir pastas como:

- `platformio/`: código-fonte do sistema embarcado;
- `scripts/`: rotinas de processamento e análise;
- `dados/`: arquivos de entrada ou conjuntos de dados de exemplo;
- `figuras/`: imagens e gráficos gerados no projeto;
- `docs/`: documentação complementar.

## Observações

Os arquivos disponibilizados neste repositório têm finalidade acadêmica e acompanham o desenvolvimento apresentado no TFC.
